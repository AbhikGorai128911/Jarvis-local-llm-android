# JARVIS V1

An offline-first terminal assistant built in Python and designed to run on Android through Termux.

## Features

- Local command execution
- App launching
- Offline-first design
- Local LLM support using llama.cpp
- Text-to-speech
- Basic memory
- Web search when online

## Requirements

- Python 3
- Termux (for Android)
- llama.cpp (optional)
- A GGUF model (optional)

## Notes

The repository does not include llama.cpp binaries or GGUF models.
